Kakao Maps Setup

카카오 JavaScript 키 입력

최초 실행 시 키를 한 번 입력하면 로컬에 저장되고, 이후에는 자동으로 앱에 진입합니다. 저장되는 값은 지도 키만이며, 업로드한 업무 데이터는 저장하지 않습니다.

도메인 설정은 Kakao Developers에서 `file://` 또는 허용 출처 조건에 맞게 등록되어 있어야 합니다.
